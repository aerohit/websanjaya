# websanjaya

[Sanjaya](http://en.wikipedia.org/wiki/Sanjaya) for the web.

Crawls nerd websites like HN, Reddit for nerdy topics.

### How to run?

#### Start the app server

    lein run

or

    lein ring server 4000

#### Start cljs compiler

Run either:

    lein figwheel

or:

    lein cljsbuild auto
